# rocketsimu
ハイブリッドロケット用6自由度弾道シミュレーションライブラリ(v0.1.0:minimum library)

# Features
- パッケージ化(依存ライブラリは自動インストールされます)
- 6自由度弾道シミュレーション
- べき法則風モデル

# Requirements
- numpy
- numpy-quaternion
- scipy
- pandas
